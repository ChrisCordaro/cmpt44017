DFA_FINAL README

This program will help you determine your current mood. Press enter to continue through the instructions
Moods that are avaliable are 'Neutral', 'Happy', 'Sad', and 'Focused'
Accepted input includes 'h s f' \n For example: If I have felt happy the past 2 days, and sad the other three days, my input would look like this 'ssshh' (without quotes)
Enter exactly 5 inputs. The goal is to reach a happy state. If the program determines that you are not overall happy, it will recommend the type of music you should listen to
